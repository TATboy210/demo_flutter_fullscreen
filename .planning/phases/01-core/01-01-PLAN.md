    **fullscreen_window 切换方法 (per PITFALL-1, PITFALL-3, PITFALL-14):**
    ```
    Future<void> _toggleFwFullscreen() async {
      if (_fwToggling) return;  // 防抖
      _fwToggling = true;
      final previous = _fwFullscreen;
      setState(() => _fwFullscreen = !_fwFullscreen);
      try {
        await FullScreenWindow.setFullScreen(_fwFullscreen);
        // 强制布局重建（per PITFALL-3 Windows 布局修复）
        if (!_fwFullscreen && mounted) {
          WidgetsBinding.instance.addPostFrameCallback((_) {
            if (mounted) setState(() {});
          });
        }
      } catch (e) {
        if (mounted) {
          setState(() => _fwFullscreen = previous);  // 回滚
          ScaffoldMessenger.of(context).showSnackBar(
            SnackBar(content: Text('fullscreen_window 失败: $e')),
          );
        }
      } finally {
        _fwToggling = false;
      }
    }
    ```

    **flutter_fullscreen 切换方法 (per PITFALL-1, PITFALL-7):**
    ```
    void _toggleFfFullscreen() {
      if (_ffToggling) return;  // 防抖
      _ffToggling = true;
      FullScreen.setFullScreen(!_ffFullscreen);
      // 不在这里 setState —— 由 listener 确认后更新状态
    }
    ```

    **flutter_fullscreen Listener (per PITFALL-10):**
    - 在 initState 中创建自定义 FullScreenListener 实现

    - 回调中通过 mounted 检查后 setState 更新 _ffFullscreen 和 _ffToggling
    - 在 dispose 中调用 FullScreen.removeListener(_ffListener) 清理

    FullScreenListener 实现要点：
    - 使用 FullScreenListener mixin
    - onWindowEnterFullScreen: setState(() { _ffFullscreen = true; _ffToggling = false; })
    - onWindowLeaveFullScreen: setState(() { _ffFullscreen = false; _ffToggling = false; })
    - onWindowMaximize / onWindowUnmaximize: 空实现（不需要处理）
